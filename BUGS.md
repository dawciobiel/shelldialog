## Reporting Bugs

When reporting a bug, please include:

1.  A clear and concise description of the bug.
2.  Steps to reproduce the behavior.
3.  Expected behavior.
4.  Actual behavior.
5.  Screenshots (if applicable).
6.  Environment details (OS, Java version, etc.).
