# Documentation

This directory contains user-facing documentation for the current public API.

- `dialogs.md`: how to build and use `TextLineDialog`, `SingleChoiceDialog`, `MultiChoiceDialog`, `PasswordDialog`, and `YesNoDialog`, including optional builder settings such as validation and default selections
- `dialog-theme.md`: how `DialogTheme` works and which style is used by each UI area
- `dialog-options.md`: how `SimpleDialogOption` and `DialogOption` are used
- `publishing.md`: how the project is prepared for Maven Central publication and which prerequisites are required
- `ui-areas.md`: how to configure `TitleArea`, `ContentArea`, `InputArea`, and `NavigationArea`

The examples in `src/main/java/io/github/dawciobiel/shelldialog/examples` are the best companion to these documents.
